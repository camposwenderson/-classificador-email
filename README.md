# Classificador de Emails - Projeto de Automação

## Descrição

Este projeto tem como objetivo **automatizar a leitura e classificação de emails** para uma grande empresa do setor financeiro, permitindo que a equipe se concentre em tarefas importantes, enquanto a solução sugere respostas automáticas para emails recebidos.

O sistema classifica emails em **Produtivo** ou **Improdutivo** e sugere respostas automáticas adequadas com base na classificação.

**Categorias de Classificação:**
- **Produtivo:** Emails que requerem ação ou resposta específica.
- **Improdutivo:** Emails que não necessitam de ação imediata.

---

## Funcionalidades

- Upload de emails em **.txt**, **.pdf** ou inserção de texto manual.
- Classificação automática de emails em **Produtivo** ou **Improdutivo**.
- Geração de respostas automáticas sugeridas para cada email.
- Interface web simples e intuitiva para fácil uso.
- Integração com AI para processamento de linguagem natural.

---

## Tecnologias Utilizadas

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Python, Flask/FastAPI
- **Processamento de Linguagem Natural:** Hugging Face Transformers, NLTK, spaCy
- **Hospedagem:** Heroku / Render / Hugging Face Spaces (substitua conforme usado)
- **Controle de Versão:** Git e GitHub
